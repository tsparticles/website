# Plan 01: Playground & Preset Library

Objective: Implement an editable, client-side playground that loads presets from `configs/`, lets users live-edit parameters with immediate rendering (debounced), and supports copyable JSON + client-side shareable URLs.

Scope (must deliver for Phase 1):

- Implement playground shell and editor controls mapped to preset schema.
- Preset library UI with one-click load and lazy preview thumbnails.
- Live-edit behavior with debounce/defaults and heavy-config fallback (Run button).
- Export: Copy JSON and client-side shareable URL (LZ-String fragment encoding) with size fallback.
