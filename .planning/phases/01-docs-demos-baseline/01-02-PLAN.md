# Plan 02: Docs Integration & Preview Generation

Objective: Integrate the playground into site docs, add build-time hooks for optional precomputed thumbnails, and ensure docs generation includes the preset library where appropriate.

Tasks:

1. TASK-DI-01: Docs integration points
   - Add playground entry in documentation pages and a central `/playground` page.
   - Ensure `docs-gen` copies `configs/` into doc assets or references them reliably.

2. TASK-DI-02: Optional thumbnail build step (deferred to opt-in)
   - Add a Node script `scripts/generate-presets-thumbs.js` that can be run in CI to render thumbnails using headless browser or node-canvas.
   - Add a workflow job (optional) to run this script and store thumbnails under `docs-assets/presets/`.

3. TASK-DI-03: CI docs check
   - Add a CI check to ensure docs build succeeds and playground assets are present. Fail CI if docs build fails.

Must haves:

- Docs build includes playground entry and loads presets.
- Optional thumbnail step documented and runnable.

Verification:

- Run `npm run docs` or the docs generation step in CI and ensure no errors.

Estimated effort: 2-4 days.
